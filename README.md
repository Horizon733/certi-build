# Certi-builder
A simple commandline app for generating bulk of certificates
# Installation
## Using Pip
```bash
  $ pip install certi-builer
```
## Manual
```bash
  $ git clone https://github.com/Horizon733/certi-build
  $ cd certi-build
  $ python setup.py install
```
# Usage
```bash
$ certi_builder build
```

# bold text
```bash
$ certi_builder build --bold
```
